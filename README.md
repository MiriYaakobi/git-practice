# Git Practice Lab

This repository is dedicated to learning and practicing advanced Git and GitHub workflows.

## Overview
This project serves as a sandbox to experiment with various Git commands and scenarios without fear of breaking production code.

## Concepts practiced, for example:
* **Rewriting history:** `git rebase -i` and compressing commits.
* **Cherry picking:** Applying specific commits across branches.
* **Storage:** Temporarily storing unfinished work.
* **Debugging:** Using `git bisect` and `git reflog`.
* **Collaboration:** GitHub Flow rules, Pull Requests, and Branch Protection.

Created for educational purposes.
